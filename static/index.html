Australia Wide Emergency Map

Australian live incident map covering all states and territories.

Launch the Map ➞

Incident data is collated directly from: emergency.vic.gov.au - Victoria, rfs.nsw.gov.au - New South Wales, emergency.wa.gov.au - Western Australia, geohub.sa.gov.au, data.eso.sa.gov.au - South Australia, fire.tas.gov.au - Tasmania, and qfes.qld.gov.au - Queensland.

Do not use this app as a warning or alert service. Do not use this app for safety of life decisions. Use it only as a data analysis tool and for seeing a big-picture overview.

This map is in no way officially affiliated with any emergency services, state government or data source.

For local updates and alerts, please refer to your state emergency or fire service.

This is a non-commercial, volunteer-time project. Source code is available and feedback or contributions are welcome.

The layout works best on desktop, but it is also great on tablet or mobile where it can be added to the home screen.

For an overview of features please see the User Guide.

Built with free, open-source, community-driven software: Leaflet, OpenStreetMap, Vue.js and Firefox.
